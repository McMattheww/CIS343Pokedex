# Pokedex

Program built with node.js utilizes the API at https://pokeapi.co/ to allow the user to search for information about Pokemon

# Run Instructions
* Ensure Node.js is installed, use command "node --version"
* If your node version is 17 or lower, you also need to install node-fetch. Look up how to install node-fetch on your system
* Navigate to folder with main.js and run command "node main.js"
